# MicroModel
micro model in es6 for client.

## Getting Started

```
$ npm install micro-model
```

```javascript
import MicroModel from 'micro-model';
```

## API

- defaults
- set
- get
- dispatchChange
- addChangeListener
- removeChangeListener

## Options

```javascript
let options = {
  defaults: {},
  localStorage: ture, // bool(default: true)
};

new SomeModel(options);
```
